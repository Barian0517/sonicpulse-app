# SonicPulse Visualizer 🎵

![SonicPulse Visualizer](public/assets/banner.png)

SonicPulse 是一款為桌面端打造的高性能音訊視覺化工具，將您的音樂與系統聲音轉化為炫目的動態視覺藝術。

## ✨ 特色功能

- **多樣化視覺效果**：支援 圓形 (Circle)、球面 (Sphere)、線條 (Line) 等多種幾何形狀。
- **全方位音訊監聽**：
  - **檔案播放**：支援 MP3, WAV, FLAC 等格式，具備多曲目播放清單。
  - **麥克風輸入**：即時捕捉您的聲音或外部音源。
  - **系統音訊**：擷取桌面聲音（需在分享視窗時勾選「分享音訊」）。
- **高度自定義**：自由調整顏色、條柱數量、長度、粗細、震動感及背景圖片。
- **懸浮窗模式 (Overlay)**：
  - 獨立的置頂透明視窗。
  - 自動記憶視窗位置、大小與鎖定狀態。
  - 支援「滑鼠穿透」鎖定，完美融入桌面背景。
- **粒子特效**：內建 雪花 (Snow)、下雨 (Rain)、櫻花(Sakura)等背景粒子。
- **即時錄影**：支援將視覺化畫面連同音訊錄製為高品質 WebM 影片。(beta)

## 🚀 使用指南

### 1. 開始使用
- **導入音樂**：點擊「開啟檔案」圖示，選取一首或多首音樂檔案。
- **切換模式**：在控制面板中切換「麥克風」或「系統音訊」監聽。(系統音訊使用瀏覽器畫面分享協議，部分裝置可能無法取的聲音)

### 2. 自定義外型
- **顏色**：支援單色、漸層或隨機顏色。
- **形狀**：從下拉選單選擇不同的渲染形狀。
- **背景**：可上傳自己的圖片，並調整背景震動感與透明度。

### 3. 懸浮窗模式 (桌面裝飾)
- 點擊「開啟懸浮窗」。
- 在「編輯模式」下，您可以隨意拖曳、縮放懸浮窗。
- 調整完成後點擊「鎖定」，懸浮窗將會隱藏邊框並支援滑鼠穿透，成為桌面背景的一部分。

### 4. 錄製功能(beta)
- 點擊「錄製」按鈕開始擷取畫面。
- 若使用檔案播放模式，點擊「自動渲染」會從頭播放並錄製整首歌曲。

## 🛠️ 系統需求
- **Windows**: 支援 Windows 10/11 (**穩定版本 - 推薦使用**)
- **其他平台**: macOS 與 Linux 版本目前仍處於實驗性階段，運作可能不穩定。
