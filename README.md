# 櫻撥放器 (Sakura Player) 🌸🎵

![Sakura Player](public/assets/banner.png)

**櫻撥放器 (Sakura Player)** 是一款基於 Tauri 打造的全方位跨平台桌面音樂播放與音訊視覺化工具。它不僅提供沉浸式的視覺饗宴，更整合了強大的多來源音樂播放能力，將您的桌面變成專屬的音樂派對中心。

## ✨ 核心特色與功能

### 🎵 1. 全能音樂播放與來源整合
櫻撥放器支援多種音樂來源，滿足您所有的聽歌需求：
- **本地音樂庫 (Local Library)**：自動掃描並建立您的本地音樂收藏，支援各種常見音訊格式，並可輕鬆管理播放清單。
- **網易雲音樂 (Netease Cloud Music)**：深度整合網易雲 API，支援手機 App 掃碼登入。享受您的紅心歌曲、每日推薦、建立歌單，甚至具備智慧的「相似歌曲漫遊 (Roaming)」功能，讓好音樂源源不絕。
- **Navidrome 串流**：連接您個人的 Navidrome 伺服器，隨時隨地串流播放專屬雲端音樂庫。
- **MusicFree 插件系統(暫停維護)**：強大的插件管理器，支援直接安裝 MusicFree 插件，實現跨平台智慧搜尋，想聽什麼就搜什麼。*(備註：目前存在許多問題，僅測試 bilibili、網易雲、audiomack 插件正常運作)*

### 📻 2. 獨創「線上點歌系統」 (Jukebox)
內建輕量化網頁伺服器，開啟後可以讓同一區網內的親朋好友或觀眾透過手機 / 電腦網頁直接向播放器點歌！
- **權限控制**：可設定是否允許他人切歌、插播、暫停或管理序列。
- **個人模式**：開啟後點歌系統將可存取主機的歷史紀錄與自訂歌單。
- **多來源搜尋**：點歌頁面同樣支援網易雲、Navidrome 與 MusicFree 的智慧搜尋。

### 🎨 3. 沉浸式音訊視覺化 (Visualizer)
將您的音樂轉化為炫目的動態視覺藝術：
- **多樣化幾何形狀**：支援 Grid 3D、圓形 (Circle)、球面 (Sphere)、線條 (Line) 等多種視覺特效。
- **全方位音訊監聽**：除了播放器內建音樂，也支援**麥克風輸入**與**系統音訊**的即時聲音擷取。
- **背景與粒子特效**：可自訂背景圖片，並內建櫻花 (Sakura)、雪花 (Snow)、下雨 (Rain) 等浪漫粒子特效。
- **動態歌詞**：支援顯示即時動態歌詞，可自訂字型、入場動畫、循環動畫、發光，等字幕效果，跟隨音樂律動。

### 🖥️ 4. 懸浮窗模式 (Overlay / Desktop Widget) (暫停維護)
- 獨立的置頂透明視窗，可自由拖曳與縮放。
- 支援「鎖定與滑鼠穿透」，讓視覺化特效完美融入您的桌面背景，不干擾日常工作。

### 🎥 5. 錄影與匯出 (暫停維護)
- 內建高品質畫面與音訊捕捉功能，一鍵將酷炫的視覺化過程錄製為 WebM 影片與朋友分享。

### 🌍 6. 多語系支援 (i18n)
- 介面提供繁體中文 (zh-TW)、簡體中文 (zh-CN)、英文 (en)、日文 (ja) 自由切換。

## 🚀 使用指南

### 快速開始
1. **選擇音樂來源**：在左側選單選擇「本地音樂」、「網易雲」或「Navidrome」開始聆聽。
2. **自訂視覺特效**：點擊右上角的設定 / 視覺化選單，調整頻譜形狀、顏色漸層與背景圖片。
3. **開啟懸浮窗**：如果您希望將特效作為桌面裝飾，點擊「懸浮窗」按鈕並設定為「滑鼠穿透」。
4. **與朋友同樂**：進入「設定 > 線上點歌 (Jukebox)」，啟用伺服器並將網址分享給朋友，讓他們為您點歌。

## 🛠️ 技術架構
- **Core**: Tauri v2 (Rust)
- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS / Vanilla CSS
- **Audio Processing**: Web Audio API

## 💻 系統需求
- **Windows**: 支援 Windows 10/11 (**穩定版本 - 推薦使用**)
- **其他平台**: macOS 與 Linux 版本可編譯運行，但因系統音訊擷取 API 差異，部分功能可能受限。
